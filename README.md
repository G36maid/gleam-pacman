# pacman

A Pac-Man implementation in Gleam  
using the Tiramisu game engine.

Documentation: [Tiramisu HexDocs](https://hexdocs.pm/tiramisu/)

[![Package Version](https://img.shields.io/hexpm/v/pacman)](https://hex.pm/packages/pacman)
[![Hex Docs](https://img.shields.io/badge/hex-docs-ffaff3)](https://hexdocs.pm/pacman/)

## Installation

Add Pac-Man to your Gleam project:

```sh
gleam add pacman@1
```
